# GoldiT

Ce projet simple consiste à afficher durant quelque secondes une grilles contenant des lettres qui vont par paires
après ça, les cases deviennent noires
l'utilisateur doit cliquer sur 2 cases à la suite pour essayer de trouver les paires

quand l'utilisateur a trouvé, le programme affiche le nombre de tentatives qu'il lui a fallu pour terminer la grille
